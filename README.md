# Evaluaci-n-preanest-sica
La plataforma web definitiva para el anestesiólogo moderno. Centraliza pesos ajustados farmacocinéticos, predictores de vía aérea y escalas críticas (Lee, Caprini, Arné, Apfel, STOP-BANG) en una sola interfaz interactiva. Optimiza tus decisiones clínicas en quirófano a un solo clic y en tiempo real. 🩺
